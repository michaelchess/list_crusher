List Crusher:

List Crusher is a tool to determine whether a set of de novo mutations falls on a set of the genome more than would be expected by chance.
The program takes a list of mutations and a list of genes of interest and returns a set of statistical analyses.

Please be sure that your mutations file contains only one gene and one of its associated mutations per line
And please be sure that your area of interest file contatins a list with one gene per line
The Excel specialty format 'Tab Delimited Text (.txt)' is recommended
